# Queuing Examples

7 examples exploring queuing theory, metastability, and failure modes.

| Example | Description |
|---------|-------------|
| [m_m_1_queue.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/queuing/m_m_1_queue.py) | Classic M/M/1 queue with metastable failure demonstration |
| [metastable_state.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/queuing/metastable_state.py) | Metastable state dynamics under load |
| [gc_caused_collapse.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/queuing/gc_caused_collapse.py) | GC-induced metastable collapse |
| [retrying_client.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/queuing/retrying_client.py) | Retry amplification and thundering herd |
| [increasing_queue_depth.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/queuing/increasing_queue_depth.py) | Queue depth growth under increasing load |
| [dual_path_queue_latency.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/queuing/dual_path_queue_latency.py) | Dual-path queue latency comparison |
| [load_aware_routing.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/queuing/load_aware_routing.py) | Load-aware routing across multiple servers |

## Running

```bash
python examples/queuing/m_m_1_queue.py
```
