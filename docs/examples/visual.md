# Visual Debugger Example

| Example | Description |
|---------|-------------|
| [visual_debugger.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/visual/visual_debugger.py) | Bursty M/M/1 queue with browser-based visual debugger |

## Running

Requires the visual extras:

```bash
pip install happysim[visual]
python examples/visual/visual_debugger.py
```

Opens a browser at `http://127.0.0.1:8765` with entity graphs, charts, event logs, and playback controls.

See the [Visual Debugger Guide](../guides/visual-debugger.md) for a full walkthrough.
