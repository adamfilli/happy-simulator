# Industrial Examples

20 examples modeling real-world service and manufacturing operations.

|---------|-------------|

| [manufacturing_line.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/manufacturing_line.py) | Assembly line manufacturing |

| [grocery_store.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/grocery_store.py) | Grocery checkout with express lanes and self-checkout |
| [car_wash.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/car_wash.py) | Sequential car wash pipeline |
| [restaurant.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/restaurant.py) | Full-service restaurant with reservations and kitchen pipeline |
| [supply_chain.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/supply_chain.py) | Multi-echelon supply chain with bullwhip effect |
| [warehouse_fulfillment.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/warehouse_fulfillment.py) | Order fulfillment with batch picking and zone routing |

| [drive_through.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/drive_through.py) | Drive-through restaurant |
| [laundromat.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/laundromat.py) | Laundromat with washers, dryers, and folding tables |
| [pharmacy.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/pharmacy.py) | Pharmacy with verification rework and perishable inventory |
| [theme_park.py](https://github.com/adamfilli/happy-simulator/blob/main/examples/industrial/theme_park.py) | Theme park with rides, FastPass, and balking |
